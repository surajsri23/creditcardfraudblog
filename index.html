SwiftSecured: Detecting Credit Card Fraud with Machine Learning

With the rise of online transactions, credit card fraud has become a critical issue. SwiftSecured is a full-stack solution that utilizes machine learning to detect fraudulent activity in real time, all packed inside a responsive and secure web interface.

Tech Stack

  • Frontend: HTML, Tailwind CSS, JavaScript
  • Backend: Python Flask
  • ML Model: Trained on Kaggle Credit Card Fraud Dataset
  • Database: SQLite
  • Deployment: Render.com

How It Works

  1. User inputs anonymized transaction data.
  2. Data is sent to the Flask backend via JS.
  3. Model predicts: Fraud / Not Fraud.
  4. Result is stored in the SQLite database with timestamp.
  5. User can view a prediction history log.

Key Features

  • Live prediction with a trained ML model
  • Clean UI with responsive Tailwind CSS
  • History logging with SQLite
  • Model integration via Joblib
  • Deployed and accessible globally
